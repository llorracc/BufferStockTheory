%%%
%%% This file mirrors the boilerplate already used by Introduction.tex and
%%% Appendices/Apndx*.tex, so the same standalone-or-integrated compile
%%% mechanism applies.

% ... section content goes here ...
% ---------------------------------------------------------------------

% No standalone bibliography in section subfiles. Mirrors the HAFiscal
% Subfiles/*.tex pattern: standalone PDF shows section content with
% cross-refs (resolved via \externaldocument) but no References. The
% paper-wide bibliography is emitted from BufferStockTheory.tex (when
% the full doc compiles) or BufferStockTheory-NoAppendix.tex (when the
% body-only doc compiles). Adding a bib invocation here would also fire

% class), causing duplicate \bibdata writes that break bibtex.
